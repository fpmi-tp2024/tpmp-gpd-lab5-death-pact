Функционал приложения

Данное приложение позволяет автопарку хранить и обновлять информацию об автомашинах, водителях и выполненных заказах в базе данных. Также предоставляется возможность получать отчёты о заработанных водителями деньгах на основе отдельной таблицы базы данных

Консольное приложение предоставляет различный функционал, адаптирующийся под задачи конкретного пользователя

Для администраторов

  1. Просмотреть выполненные в какой-то период заказы конкретного водителя
  2. Просмотреть общий пробег и общую массу перевезенных грузов для конкретной машины
  3. Просмотреть общее количество поездок, общую массу перевезенных грузов, сумму заработанных денег для конкретного водителя
  4. Просмотреть все сведения и сумму заработанных денег о водителе с наименьшим количеством поездок
  5. Просмотреть все сведения о машине с наибольшим общим пробегом

Для водителей

  1. Просмотреть выполненные в какой-то период заказы
  2. Просмотреть общий пробег и общую массу перевезенных грузов для своей машины
  3. Просмотреть общее количество поездок, общую массу перевезенных грузов, сумму заработанных денег
  4. Просмотреть все сведения о своей машине

Для незарегистрированных пользователей

  1. Сделать заказ на грузоперевозку
